# Influence of Unfunded-Aid on Enrollment

Conducted a comprehensive analysis to determine the impact of unfunded aid on student enrollment, focusing on demographic factors such as sex, race, and citizenship status.
Utilized Python and pandas to filter and process data, categorizing students into those enrolled with aid and without aid.
Identified dominant sex and race within each citizenship category, providing insights into demographic trends influenced by aid offers.
Presented findings that highlighted the significant differences in enrollment patterns across various demographic groups.
